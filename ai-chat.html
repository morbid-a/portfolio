AI Chat Website

Full-stack C# chat with contextual memory and user recognition

Project Overview

This AI Chat Website is a full-stack C# application built from scratch in Microsoft Visual Studio. It combines a responsive frontend and a powerful backend to enable real-time conversation with contextual awareness. Users are remembered between sessions, greeted individually, and can interact through a clean, intuitive chat interface.

Login & Signup Security

The authentication system enforces strict validation rules to protect user accounts:

  • Unique username & email checks — no duplicates allowed.
  • Strong password enforcement — minimum of 2 uppercase, 2 lowercase, 2 numbers, and 2 special characters.
  • Real-time validation feedback during signup.
  • Login allowed with username only, but requires correct password.
  • Invalid credentials are immediately rejected.
All authentication logic is coded in pure C# with no external frameworks, and follows secure API/REST communication patterns between frontend and backend.

Context-Aware AI Chat

The chat engine maintains conversation history per user, allowing for contextual responses instead of isolated Q&A. This means the AI can reference previous messages, remember details like names or preferences, and adapt replies accordingly.

All real-time messaging is handled using a REST API interface to the backend, ensuring smooth and reliable updates without constant page refreshes.

Technology Stack

This project is fully coded from scratch in Microsoft Visual Studio, combining:

  • Backend: C# with ASP.NET Core
  • Frontend: HTML, CSS, integrated directly with C# logic
  • API: Custom REST endpoints for authentication and AI messaging
  • Database: Lightweight SQLite engine used for encrypted credential storage and persistent chat logs
  • Hosting: Azure cloud deployment
