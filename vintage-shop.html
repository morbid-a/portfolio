AI Chat Website

Full-stack C# chat with contextual memory and user recognition

Project Overview

This AI Chat Website allows real-time conversation with contextual awareness, powered by a C# backend and WebSocket communication. Users are recognized and greeted individually.

Key Features

• Contextual memory per user
• API integration for NLP
• Secure user authentication
• Responsive chat UI for desktop & mobile

Technology Stack

• Backend: C#, ASP.NET Core
• Frontend: HTML, CSS, JS
• Database: SQL Server
• Hosting: Azure
