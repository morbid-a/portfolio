Vintage Shop

Simple front-end e-commerce concept with HTML & CSS

Project Overview

This Vintage Shop website showcases a clean, elegant front-end design for an e-commerce experience. Built with accessible HTML and refined CSS interactions — the emphasis is on visual hierarchy and pleasant micro-interactions.

Home — What the project offers

The site displays what this shop has, in its menus, meals, etc. Customers can explore different categories, view detailed descriptions, check prices, and discover any special offers or seasonal items available.

Search & Menu

The menu supports category browsing, keyword search, and live filtering by cuisine, price, or dietary preference. Results update instantly as you type (debounced for performance) and display images, descriptions, and prices for each meal.

  • Category & keyword search — type "pas" to find "Pasta Alfredo" and "Spaghetti Bolognese".
  • Add to Cart buttons allow you to quickly add any meal, update quantities, or remove items without leaving the page.

Cart Feature

The cart feature keeps track of all meals you’ve selected while browsing the menu. It shows meal names, quantities, prices, and a running total, with quick options to update quantities, remove items, or proceed to checkout — all without leaving the page.
