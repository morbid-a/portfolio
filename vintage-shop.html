Vintage Shop

Simple front-end e-commerce concept with HTML & CSS

Project Overview

This Vintage Shop website showcases a clean and elegant front-end design for an e-commerce experience. It’s built entirely with HTML and CSS, focusing on aesthetics and layout rather than complex backend logic.

Shop Menu

The shop menu page lists available vintage items in a visually appealing grid layout. Each product card highlights the item’s image, name, and price, maintaining a cohesive retro color palette.

Cart System

The cart system page provides users with a simple interface to view and manage their selected items. Although it’s a static HTML/CSS prototype, it simulates the look and feel of a real e-commerce checkout process.
