Music Player Management System

React.js frontend • JavaScript backend with JSON file storage • playlists, search, profiles

Project Overview

A full-featured music management app with a React.js frontend and a lightweight JavaScript backend that stores data in JSON files. Supports creating & managing playlists, a powerful search bar, profile management, and quick-add controls. The UI is responsive and optimized for desktop and mobile.

Stack: React · JavaScript backend · JSON storage

Authentication — Signup & Login

The app provides a secure signup/login flow with real-time validation and clear error feedback:

  • Signup: unique username & email checks (no duplicates). Passwords must meet the strong complexity rules (2 uppercase, 2 lowercase, 2 digits, 2 special characters).
  • Login: users can sign in using either username or email plus password; invalid credentials are rejected immediately.
  • Backend: authentication and validation implemented in JavaScript; user data and credential hashes stored in JSON files (structured, encrypted where needed).
  • UX: real-time client-side hints (React) + server-side enforcement for security.

Search & Library

Smart search supports prefix, partial, and full-word matching. Results update as you type (debounced for performance) and include live filtering by metadata.

  • Prefix & partial matching — type "lov" to find "Why Love Me Do" and "Lovely Day".
  • Left sidebar contains playlists and actions so you can add/edit/delete any playlist from anywhere in the UI.
  • Quick-add button in the nav allowing to create songs anywhere in the UI.

Profile & Account

Profile allows editing username (unique check), updating date-of-birth, changing password (requires current password confirmation), and account deletion (server-side removal of related JSON entries).

  • Edit username & DOB — unique username validation enforced.
  • Change password with current password confirmation.
  • Logout & delete account — deleting an account removes associated playlists and user data from the JSON store.

Technology Stack

Frontend: React.js (functional components, hooks, client-side routing).
Backend: JavaScript (Node.js or similar) serving REST endpoints; application logic written in JavaScript.
Storage: Structured JSON files for persistence (users, songs metadata, playlists). Files are read/written carefully with concurrency in mind and optional lightweight encryption for credential fields.
Dev Tools: Visual Studio / VS Code, Postman for API testing, Git for version control.
