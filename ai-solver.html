AI Problem Solver UI

Full-stack C# chat with contextual memory and user recognition

Project Overview

This AI Chat Website is a full-stack C# application built from scratch in Microsoft Visual Studio. It combines a responsive frontend and a powerful backend to enable real-time conversation with contextual awareness. Users are remembered between sessions, greeted individually, and can interact through a clean, intuitive chat interface.

Below are the three UI states demonstrating the flow: prompt input → processing → result.

Prompt Input

This screen shows the prompt field where the user types questions or tasks. The UI supports multi-line prompts and quick presets to guide the model.

  • Multi-line input with examples
  • Presets and system message shortcuts
  • Clear send/submit controls

Processing / Loading

This view shows the processing and progress indicators while the backend prepares an answer. It provides feedback like "retrieving context" or "generating response" to keep users informed.

  • Progress and status hints
  • Context retrieval from memory
  • Error handling and retry options

Result & Quick Actions

The final answer is displayed clearly, with quick actions such as copy-to-clipboard and save. This lets users reuse or store generated content easily.

  • Copy to clipboard
  • Save to notes or export
  • Regenerate with small edits

What I used

I used Google Colab and Python together with API, Autogen, and Gradio.

This implements a workflow using AI agents to make a problem solver that solves everything step by step and acts like a beginner-friendly teacher.
